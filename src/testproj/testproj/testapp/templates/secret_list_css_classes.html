Sorting with CSS Classes (Django)

This example demonstrates using CSS classes for sort indicators instead of HTML entities. When a column is sorted, the anchor tag receives a CSS class (sorted-asc or sorted-desc) which can be styled with CSS pseudo-elements.

Configuration: Set SORTING_CSS_CLASS_ASC and SORTING_CSS_CLASS_DESC in your Django settings.
